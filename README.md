# TestRNJest
Test for react-native use Jest
Example for movies 


### use
`
npm test OR npm test -- -u nameComponent.js (rendering components)
`

### Mocks for test
`
npm run jacp
`

jacp => script for test mocks and configurations

## usage jacp

``` javascript
let mocksGUI = require('./Mocks/GUIMocks')(jest);
mocksGUI.init();
```
